ax.getSelectorType(str) → {string|''}

Description:

  • Get node selector string type.'#demo' is judged as 'ID' with#; '. Demo' is judged as 'class'; 'demo' uppercase judgment as 'node'; '[demo]' medium bracket package is judged as 'attr'; 'DIV P' has a space for the father and son, and the judgment is 'Nesting'; other string returns the empty value.
